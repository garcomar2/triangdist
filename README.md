# triangdist

R package for the triangular distribution.

## Installation

```r
install.packages("remotes")
remotes::install_github("garcomar2/triangdist")
```

## Usage

```r
library(triangdist)

dtriang(0.5, min = 0, max = 1, mode = 0.5)
ptriang(0.5, min = 0, max = 1, mode = 0.5)
qtriang(0.5, min = 0, max = 1, mode = 0.5)
rtriang(5, min = 0, max = 1, mode = 0.5)
```
